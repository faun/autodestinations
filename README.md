# Setup

Dependencies:

* [Moonscript](http://moonscript.org/)

Development dependencies:

* [Luarocks](http://www.luarocks.org/en/Download)
* [Busted](http://olivinelabs.com/busted/)

Install with luarocks:


    luarocks install busted

* [Grunt](http://gruntjs.com/)

## Testing:

    grunt

## Autotest:
    
    grunt watch
